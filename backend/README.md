# Configuration 
## Run app
```bash
flask run
```

## Add .env file
```bash
touch .env
```

### .env file example
```bash
DEVICE_IP="172.16.42.1"
USERNAME="pptc"
PASSWORD=""
PIN_CODE=""
```
## Pip install
### Create venv
```bash
python -m venv venv
```

### Activate venv
```bash
source venv/bin/activate
```

### install 
```bash
pip install -r requirements.txt
```
